\frame{
\frametitle{Data Structure / Quality}
\begin{itemize}
\item 2919 Observations separated into 1460 labelled training observations and 1459 unlabelled test observations to predict.
\item Target Variable: Final Sale Price in USD
\item 79 Explanatory Variables of which 36 are numeric (e.g. floor area) and 43 categoric (e.g. heating type).
\item 19 variables contain NA's 16 of which are categoric variables
\item NA usually means a variable is not applicable (Pool Quality if house has no pool) instead of missing data
\end{itemize}
}
